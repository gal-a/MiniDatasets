**Source Attribution**

This dataset is based on the Default of Credit Card Clients dataset originally published by the UCI Machine Learning Repository:
https://archive.ics.uci.edu/dataset/350/default+of+credit+card+clients
The dataset was created and contributed by Yeh, I‑C. and Lien, C‑H., as part of their research on credit risk modeling.
The data is provided here for educational and analytical purposes as part of the MiniDatasets collection.
